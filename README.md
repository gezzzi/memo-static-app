# メモアプリ

シンプルなメモ管理アプリケーション

## 機能

- メモの追加
- メモの削除
- メモの一覧表示

## 使用技術

- React
- Tailwind CSS
- PostCSS

## インストール方法

```bash
# リポジトリのクローン
git clone [リポジトリURL]

# 依存パッケージのインストール
npm install

# 開発サーバーの起動
npm start
```

## 開発環境

- Node.js
- npm

## ビルド方法

```bash
npm run build
```
